# William Lukumbi Portfolio Website

A responsive personal portfolio website showcasing the professional profile of William Lukumbi, a Business Strategist and Data Science Professional.

## 🌟 Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Dropdown menus, hover effects, and dynamic content
- **Multi-language Support**: Resume downloads in English and French
- **Portfolio Showcase**: Display of projects and expertise
- **Contact Integration**: Social media links and contact information

## 🛠 Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Custom styling with responsive breakpoints
- **Bootstrap**: Grid system and components
- **JavaScript/jQuery**: Interactive functionality
- **Isotope**: Filtering and sorting for portfolio items
- **Themify Icons**: Icon library for visual elements

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for external resources

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Hawi-Lemma/Personal-Website-Client-William_Ngoy.git
   cd WilliamNgoy
   ```

2. **Open the website:**
   - Navigate to the `public_html` directory
   - Open `index.html` in your web browser

### Deployment

This site can be deployed to any static hosting service:

- **GitHub Pages**: Enable Pages in repository settings, set source to `public_html` folder
- **Netlify/Vercel**: Connect repository and set build command to `none` (static site)
- **Traditional hosting**: Upload `public_html` contents to your web server

## 📖 Usage

- **Navigation**: Use the sticky navigation bar to jump between sections
- **Resume Download**: Click the "Download Resume" button 
- **Portfolio**: Browse projects 
- **Contact**: Use social media links or contact form

## 🎨 Customization

### Colors
The site uses CSS custom properties (variables) for easy theming:
- `--dark-green`: Primary dark color
- `--green`: Background color
- `--yellow`: Accent color
- `--gray`: Secondary text color

### Content
Edit the following files to customize content:
- `public_html/index.html`: Main content and structure
- `public_html/assets/css/styles.css`: Styling and layout
- `public_html/assets/js/script_general.js`: JavaScript functionality
